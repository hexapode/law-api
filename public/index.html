An API to the law

Usage

GET /ask/:n/:query

:n is the number of results you want to get back.
:query is the query you want to ask the API.

GET /ask/:n/:legislation/:query

:n is the number of results you want to get back.
:query is the query you want to ask the API.
:legislation The target legislation.

POST /query

body is a JSON object with the following structure:
{ "query": "your query", "n": "number of results you want to get back", "legislation" : "the target legislation (default: US Code)" }

Available Legislations

  • US Code: use "USA Code"
  • US Code of Federal Regulation: use "USA Code of regulation"
  • USA California: use "California Legislation""
